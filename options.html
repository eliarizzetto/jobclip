Data Destination

Google Sheet appends rows to an online spreadsheet (requires the one-time Google Cloud setup below). Local CSV saves directly to a file on your computer — no account needed, nothing leaves your browser.

The part of the sheet URL between /d/ and /edit:
https://docs.google.com/spreadsheets/d/ID_HERE/edit

If a sheet with this name doesn't exist, it will be created automatically.

Saved file: No file chosen

Pick a CSV file (a new one is fine — the column header row is created on the first save). The same file is appended to on every save. The first save after starting Chrome may ask once for permission to access the file: accept to continue.
